    ```bash

    ```

    ```bash

    ```

    ```bash
    npm start
    ```

    ```bash
    cd ../frontend
    ```

6. Install frontend dependencies:
    ```bash
    npm install
    ```

7. Start the frontend development server:
    ```bash
    npm run dev
    ```
